# Установка, запуск

`docker-compose up`

Нужен свободный 3306 и 3000 порты для mysql и бэкэнда соответственно.

## Запуск тестов

`yarn test`

# Эндпоинты

### GET /calc
Первоначальный расчёт, выделил в отдельный эндпоинт
### GET /rating
Выдача счетов, отсортированных по балансу подписок
### POST /subscribe
Cоздание подписки
### POST /unsubscribe
Удаление подписки
